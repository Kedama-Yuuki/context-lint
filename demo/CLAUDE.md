# Project Instructions

## Overview

This project uses TypeScript and React. Always use TypeScript for all components.

## Rules

Use appropriate naming conventions as needed.
Never use TypeScript for test files.

Don't use inline styles.
Never modify global state directly.
Do not skip writing tests.
Avoid using var declarations.
Must not use eval or Function constructor.
